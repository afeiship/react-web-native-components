# description

> Checkbox.

## installation

```bash npm2yarn
npm install @jswork/react-input
```

## usage

> 原生 input。

```tsx live
<React.Fragment>
  <p>
    <label>
      <strong> username: </strong>
      <ReactInput type="text" onChange={(e) => console.log(e.target.value)} />
    </label>
  </p>
  <p>
    <label>
      <strong> email: </strong>
      <ReactInput type="email" onChange={(e) => console.log(e.target.value)} />
    </label>
  </p>
  <p>
    <label>
      <strong> password: </strong>
      <ReactInput type="password" onChange={(e) => console.log(e.target.value)} />
    </label>
  </p>
  <p>
    <label>
      <strong> age: </strong>
      <ReactInput type="range" onChange={(e) => console.log(e.target.value)} />
    </label>
  </p>
</React.Fragment>
```
