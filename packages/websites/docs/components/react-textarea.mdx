# description

> Checkbox.

## installation

```bash npm2yarn
npm install @jswork/react-textarea
```

## usage

> 普通的 Textarea, 不过，可以自由 resize.

```tsx live
<ReactTextarea
  autosize
  style={{ width: "100%" }}
  defaultValue={`道可道，非常道；名可名，非常名。
无名，天地之始，有名，万物之母。
故常无欲，以观其妙，常有欲，以观其徼。
此两者，同出而异名，同谓之玄，玄之又玄，众妙之门。`}
  onChange={(e) => console.log(e.target.value)}
/>
```
