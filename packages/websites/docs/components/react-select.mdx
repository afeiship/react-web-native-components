# description

> Checkbox.

## installation

```bash npm2yarn
npm install @jswork/react-select
```

## usage

> 普通的 checkbox ，适合 toggle 的场景。

```tsx live
<div>
  <label>
  <strong>Multiple: false</strong>
    <ReactSelect
      items={[
        { value: "v1", label: "label1" },
        { value: "v2", label: "label2" },
        { value: 3, label: "label3 is number" },
        { value: true, label: "label4 is bool:true" },
      ]}
      onChange={(e) => console.log(e.target.value)}
    />
  </label>

  <label>
  <strong>Multiple: true</strong>
    <ReactSelect
      multiple
      items={[
        { value: "v1", label: "label1" },
        { value: "v2", label: "label2" },
        { value: 3, label: "label3 is number" },
        { value: true, label: "label4 is bool:true" },
      ]}
      onChange={(e) => console.log(e.target.value)}
    />
  </label>
</div>
```
