# description
> Checkbox.

## installation
```bash npm2yarn
npm install @jswork/react-checkbox
```

## usage
> 普通的 checkbox ，适合 toggle 的场景。

```tsx live
<label>
  <ReactCheckbox onChange={e=>console.log(e.target.value)} />
  <strong>I agree</strong>
</label>
```
