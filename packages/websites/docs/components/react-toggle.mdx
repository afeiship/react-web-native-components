# description

> React toggle.

## installation

```bash npm2yarn
npm install @jswork/react-toggle
```

## usage

> 普通的 Textarea, 不过，可以自由 resize.

```tsx live
<ReactToggle value={true} onChange={(e) => console.log(e.target.value)} />
```
