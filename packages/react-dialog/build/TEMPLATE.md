## installation
```shell
npm install -S @jswork/react-dialog
```

## usage
1. import css
  ```scss
  @import "~@jswork/react-dialog/dist/style.css";

  // or use sass
  @import "~@jswork/react-dialog/dist/style.scss";

  // customize your styles:
  $react-dialog-options: ()
  ```
2. import js
  ```js
__GENERATE_DAPP__
  ```
